# Chocolate-Sales-Analysis-Using-Tableau

The objective of this project is to analyse Chocolate Sales data using Tableau to identify key business insights related to sales performance, product demand, geographic distribution, and sales representative efficiency.

Tools & Technologies:

📊 Microsoft Excel:-

•	Used for initial data storage and dataset preparation

•	Performed basic data cleaning (handling blanks, formatting columns)

•	Structured raw sales data into analysis-ready format

📈 Tableau:-

•	Connected Excel dataset to Tableau for visualization

•	Created interactive dashboards and story points

•	Built KPI cards (Total Orders, Total Sales, Boxes Shipped, Average Sales per Order)

•	Designed bar charts, map visualizations, and Top N analysis

•	Implemented filters (Year, Product, Country, Sales Person) for dynamic analysis

•	Applied calculated fields and aggregations for business insights

Data Source:

The dataset used for this project is the Chocolate Sales Analysis dataset, sourced from Kaggle Datasets. It contains 3,282 rows.
The dataset includes key fields such as: Sales Person, Country, Product, Date, Amount & Boxes Shipped

Problem Statement:

•	Difficulty in identifying top-performing products

•	Limited visibility into country-wise sales performance

•	Difficulty in identifying high and low performance in several fields

•	Lack of year-wise analysis to understand sales growth trends and performance changes over time

•	No structured way to analyse shipping volume distribution across regions

Walkthrough of Key Visuals:

KPI’s: Total Orders, Total Boxes Shipped, Total Sales Amount & Average Sales Per Order

Slicers: Year of Date, Product, Category & Sales Person

Bar Chart: Top 10 Products vs Sales Amount

Map Visual: Country vs Sales Amount

Bar Chart: Sales Person vs Sales Amount

Horizontal Bar Chart: Country vs Boxes Shipped

Line Chart: Date (Year) By Sales

Scatter Plot: Sales By Boxes Shipped

Project Development Process:

•	Created individual Worksheets for detailed analysis

•	Designed an interactive Dashboard by combining multiple worksheets

•	Developed a Story by integrating Worksheets and Dashboard to present insights in a structured flow

This project aims to solve these challenges by building an interactive Tableau Dashboard and Story that transforms raw sales data into meaningful business insights, enabling stakeholders to monitor performance, identify trends and make strategic decisions.
